# Evaluación Comparativa de Técnicas de Gaussian Splatting para la Digitalización de Patrimonio Cultural

## Descripción del Proyecto

Gaussian Splatting es una técnica de rasterización que permite la reconstrucción 3D y el renderizado en tiempo real de escenas fotorrealistas a partir de conjuntos limitados de imágenes. Utiliza millones de primitivas gaussianas 3D para representar escenas de manera volumétrica, logrando un balance óptimo entre calidad visual y eficiencia computacional.

En el ámbito de la preservación del patrimonio cultural, esta técnica permite crear réplicas digitales detalladas usando únicamente fotografías RGB estándar, preservando texturas finas y detalles arquitectónicos críticos para archivos digitales interactivos.

Este proyecto realizará una **evaluación comparativa exhaustiva** de implementaciones existentes de Gaussian Splatting aplicadas a la digitalización de patrimonio cultural. El estudiante creará un *dataset* propio de objetos o espacios con características relevantes y representativas del patrimonio cultural, y utilizará este conjunto de datos para evaluar múltiples implementaciones bajo criterios técnicos rigurosos:

* Velocidad de entrenamiento
* Consumo de memoria y energía
* Requisitos de hardware
* Tiempo de renderizado
* Tamaño de almacenamiento del modelo

Además de las métricas cuantitativas, se llevará a cabo un **análisis cualitativo** centrado en la preservación de detalles críticos como texturas finas y elementos arquitectónicos específicos.

El proyecto culminará con **recomendaciones prácticas** sobre qué implementación resulta más adecuada dependiendo del escenario de aplicación, considerando factores como presupuesto, hardware disponible y tipo de objeto a digitalizar. Finalmente, el *dataset* creado será documentado y publicado como un recurso adicional para la comunidad investigadora.
